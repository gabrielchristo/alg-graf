Determinando Grau de Saída dos Vértices de um Grafo

Descrição

Iremos descrever um algoritmo simples para determinar o grau de saída de cada vértice em um dado grafo.

A Ideia do Algoritmo

A ideia é iterar pela lista de adjacências de cada vértice do grafo, incrementando um contador que será o grau de saída desse vértice

Algoritmo: Grau_de_Saida

Entrada: Grafo G=(V, E)
Saída: g: V(G) → [n] vetor de vértices, onde g(v) é o grau de v em G.
  1. Para todo v ∈ V(G):
  2.     g(v) ← 0;
  3.     Para todo w ∈ Adj(v):
  4.         g(v) ← g(v) + 1

Complexidade

Complexidade de Espaço: O(n+m)
Complexidade de Tempo: O(n+m)
